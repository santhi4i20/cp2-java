# Projeto CP2 - DtoValidationBasic

## Descrição
Projeto desenvolvido para o Checkpoint 2 da disciplina Java Avançado, utilizando Spring Boot, DTO, validações e persistência em banco Oracle.

## Tecnologias utilizadas
- Spring Boot 3.2.5
- Spring Data JPA
- Hibernate
- Oracle Database
- Java 21
- Maven
